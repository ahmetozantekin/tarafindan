# TARAFINDAN
### evet tarafından
yet another open source license generator.

![demo](https://i.ibb.co/KsYM94L/record.gif)

## Installation

Install with [npm](https://www.npmjs.com/package/tarafindan) or [yarn](https://github.com/yarnpkg/yarn).

```
npm i -g tarafindan
yarn global add tarafindan
```

## Usage

deadly simple.

```bash
> tarafindan
```
